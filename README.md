# WordCloud

## 사용법

1. 리뷰 데이터를 json 형태로 input에 넣습니다.
2. main.py의 file_name을 해당하는 json 파일 이름으로 수정합니다.
3. 파일을 실행합니다.

---

## 테스트용 데이터

- 샘플 리뷰 데이터가 현재 input 폴더에 들어있습니다.
- 동작 테스트를 위해서는 result 폴더를 비워주시고 실행해주시면 됩니다.

---
